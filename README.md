# vk_music_parse
Парсим музыку со страницы ВК
Самый простой вариант следуйте инструкции из консоли
Когда дойдете до этапа подтверждения входа то войдите в свой аккаунт ВК
Далее откройте или список всех ваших песен то ссылка будет похода на ту что в консоли
Либо же перейдите в музыку человека (если открта)
Далее пролистайте сразу все треки до конца страницы, что бы прогрузить их
Нажмите энтер в скрипте и ждите
