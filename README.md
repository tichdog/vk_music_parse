# vk_music_parse

Парсинг музыки со страницы ВКонтакте

## Использование

1. Запустите скрипт и следуйте инструкциям в консоли
2. Когда дойдете до этапа подтверждения входа - войдите в свой аккаунт ВК
3. После входа:
   - Либо откройте список всех ваших песен (ссылка будет похожа на ту, что в консоли)
   - Либо перейдите в музыку человека (если она открыта)
4. Пролистайте все треки до конца страницы, чтобы они прогрузились
5. Нажмите Enter в скрипте и ждите завершения
